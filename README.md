Flutter engine development tool.

### Installation

1. Clone this repository
2. Add `crank/bin` to your `PATH`

### Examples

Build the Flutter engine:

```sh
crank build --clean --fetch
```

Run tests:

```sh
crank test
```

Run a Flutter app using the locally built engine:

```sh
crank run
```
